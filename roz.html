Road of OZ

Road of OZ is a mobile step-tracking app built with React Native and Expo that turns everyday walking into an immersive journey. Users can choose themed adventures inspired by stories like The Lord of the Rings or real-world treks, progressing through landmarks as they walk in real life.

The app tracks steps in real time using the device pedometer, even in the background, and rewards progress with unlockable events, notifications, and detailed journey stats to keep users motivated.

Key Features

  • Real-time step tracking using device pedometer (works in background)
  • Immersive journey system with themed routes and step goals
  • Landmark unlocking system with notifications and interactive popups
  • Animated step counter and progress tracking
  • 7-day stats, distance tracking, and calorie estimation
  • Adjustable step sensitivity to filter inaccurate motion

Technologies Used

Frontend: React Native (Expo)

State Management: React Context API

APIs & Libraries: expo-sensors, expo-notifications, AsyncStorage

Data Visualization: Victory Native, react-native-svg
